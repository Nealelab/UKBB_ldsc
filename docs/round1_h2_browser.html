Results from the Neale Lab

Last updated 2019-10-19


Note

You are viewing results from partitioned LD score regression with the Round 1 GWAS. We recommend using the current results instead.


Lookup a phenotype

Explore results of partitioned LD score regression for UK Biobank phenotypes. You can read more about this analysis and find links to more information on the annoucement page.

You can search for phenotypes by:

  • Description, e.g. “height”
  • ICD code, e.g. “I25”
  • ID, i.e. the UK Biobank phenotype ID matching the UK Biobank showcase

Results shown here include:

  • N: The sample size in the UK Biobank analysis set (more info on the GWAS analysis here)
  • Prevalence (Prev.) for binary phenotypes
  • Intercept (Int.): the intercept term from partitioned LD score regression. Values > 1 may indicate population stratification, confounding, or some other model misspecification. P-value (Int. p) corresponds to the test that the intercept is greater than 1.
  • Heritability (\(h^2\)): the SNP heritability estimate from partitioned LD score regression. Value is given on the liability scale for binary phenotypes, assuming that the population prevalence matches the prevalence in the UK Biobank analysis set. P-value (h2 p) corresponds to the test that the intercept is greater than 1.
