Lookup a phenotype

Explore results of univariate LD score regression for UK Biobank phenotypes. You can read more about this analysis and find links to more information on the annoucement page.

You can search for phenotypes by:

  • Description, e.g. height
  • ICD code, e.g. I25
  • ID, i.e.the UK Biobank phenotype ID matching the UKBB showcase

Results shown here include:

  • N: The sample size in the UKBB analysis set (more info on the GWAS analysis here)
  • Prevalence (Prev.) for binary phenotypes
  • Intercept (Int.): the intercept term from univariate LD score regression. Values > 1 may indicate population stratification, confounding, or some other model misspecification. P-value (Int. p) corresponds to the test that the intercept is greater than 1.
  • Heritability (\(h^2\)): the SNP heritability estimate from univariate LD score regression. Value is given on the liability scale for binary phenotypes, assuming that the population prevalence matches the prevalence in the UK Biobank analysis set. P-value (h2 p) corresponds to the test that the intercept is greater than 1.
